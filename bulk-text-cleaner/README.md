# Массовая очистка текстов рассказов

Скрипт обходит папку `all_stories`, в каждой подпапке находит `.txt` (кроме уже созданного `clean_text.txt`), очищает текст и сохраняет результат в `clean_text.txt` в той же папке.

## Установка

```bash
pip install -r requirements.txt
```

Или только tqdm:

```bash
pip install tqdm
```

## Структура

- Рядом со скриптом должна лежать папка **all_stories**.
- Внутри `all_stories` — подпапки (около 2000), в каждой один `.txt` с «грязным» текстом.
- После запуска в каждой подпапке появится **clean_text.txt** (оригинал не трогается).

## Запуск

**Вариант 1 — окно (рекомендуется):**

- **Двойной щелчок по `Запуск_очистки.bat`** — откроется окно без терминала. (Один раз выполни `pip install tqdm`.)
- Или из терминала: `cd bulk-text-cleaner`, `pip install tqdm`, `python clean_stories_gui.py`.

В окне: выбери папку, в которой лежат подпапки с .txt (кнопка «Обзор…»), нажми **Старт**. Внизу появится отчёт.

**Вариант 2 — из консоли:**

```bash
cd bulk-text-cleaner
pip install tqdm
python clean_stories.py
```

Если `all_stories` лежит в другом месте — в GUI нажми «Обзор…» и выбери папку, либо в консоли:

```bash
python clean_stories.py "D:\путь\к\all_stories"
```
