# roadmap.md — Портал аудиорассказов 18+

## 0) Минимальный функционал (что должно работать в итоге)
- [ ] Публичная часть сайта:
  - [ ] Главная (витрины: новое/популярное/категории)
  - [ ] Каталог (категории + фильтры + сортировка)
  - [ ] Поиск по названию/тегам
  - [ ] Страница рассказа (обложка/описание/теги + аудиоплеер)
- [ ] Аудиоплеер:
  - [ ] play/pause
  - [ ] перемотка
  - [ ] скорость (опционально, но лучше сразу)
  - [ ] сохранение прогресса прослушивания (для авторизованных или в localStorage)
- [ ] Админ-панель (минимум):
  - [ ] логин админа
  - [ ] создать/редактировать/удалить рассказ
  - [ ] загрузить аудио-файл и обложку
  - [ ] назначить категории/теги
  - [ ] опубликовать/скрыть рассказ
